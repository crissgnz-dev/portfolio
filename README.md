# Cristian Gonzalez - Fullstack Developer Portfolio

Bienvenidos al código fuente de mi portafolio personal web.

## 🚀 Tecnologías

- **React 19** & **Vite**: Para una experiencia frontend rápida, moderna y de alto rendimiento.
- **Tailwind CSS 4**: Para un diseño estilizado, responsive y profesional.
- **Three.js** & **React Three Fiber (Drei)**: Para integrar elementos interactivos y modelos 3D increíbles.
- **Framer Motion**: Para animaciones fluidas y transiciones dinámicas.
- **Lucide React**: Para iconos modernos y limpios.

## ✨ Características Especiales

- **Hero Interactivo 3D**: Una introducción visual impactante con modelos 3D que reaccionan al entorno.
- **Diseño Responsive**: Optimizado a la perfección para móviles, tablets y monitores de escritorio.
- **Demostración de Proyectos**: Presentación clara y atractiva de trabajos recientes o destacados.
- **UX/UI Moderno**: Estética cuidada, con modo oscuro, acentos en neón y paletas de colores armónicas.

## 🛠️ Despliegue Local

Para correr este proyecto en tu propia máquina, sigue estos pasos:

1. **Clona el repositorio:**

   ```bash
   git clone https://github.com/tu-usuario/portfolio.git
   ```

2. **Entra al directorio:**

   ```bash
   cd portfolio
   ```

3. **Instala las dependencias:**

   ```bash
   npm install
   ```

4. **Inicia el servidor de desarrollo:**

   ```bash
   npm run dev
   ```

5. **Abre tu navegador:**
   Visita `http://localhost:5173`
